# Simulation 1 CVC 细化规则记录

## 默认实验设置（当前实现）

- 5-fold CV。
- CVC 参数：`alpha = 0.05`（单一默认值）、`alpha' = 0.005`、`B = 200`。
- 候选模型：含截距的 16 个子模型。
- 对比方法：`cvc_pmax`、`cvc_frac`、`cv`、`bic`、`fy`。

## 你关心的并列最简模型（tie）处理

记 `A_cv = {m : p_hat_cv,m >= alpha}`。

1. 先找最简模型集合：
`M_min = {m in A_cv : |J_m| = min_{k in A_cv} |J_k|}`。

2. 两种 CVC 口径：
- `cvc_pmax`：从 `M_min` 中选 `p_hat_cv,m` 最大的那个作为最终模型。
  若仍并列，则按更小 CV 风险、再按更小模型编号打破平局。
  正确率按 0/1 计。
- `cvc_frac`：不再强行单选；若 `|M_min| = K`，且真模型在 `M_min` 中，
  该次重复记正确率 `1/K`，否则记 `0`。
  即“并列最简模型平均分摊正确率”。

## 正确率统计口径

- `cv`、`bic`、`cvc_pmax`：每次重复是 0/1，最终对重复取均值。
- `cvc_frac`：每次重复可取分数值（例如 0.25），最终对重复取均值。

## 备注

- 若某次 `A_cv` 为空，按实现约定回退到 `m_cv`，保证可选模型非空。
- `fy` 目前采用可落地近似实现：对每个子模型与全模型做嵌套 F 检验，
  不拒绝（`p >= alpha`）的模型进入 FY 置信集；并记录其集合大小。
