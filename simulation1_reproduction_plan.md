# Simulation 1 复现计划（R 2025.09.2+418 适配版）

## 0. 目标与成功标准
- 目标：复现论文 Simulation 1（线性模型子集选择一致性）。
- 成功标准：
  1. 在 `n = 40,80,160,320,640` 下，CVC 的正确选模率随样本量上升并接近 1。
  2. 标准 CV 保持过拟合倾向，正确选模率不随 n 同步逼近 1。
  3. `α = 0.01, 0.05, 0.1, 0.2` 的趋势与论文 Table 1 一致。

## 1. 数据与模型设定冻结
- 采用论文设定：5 维协变量（含截距）线性模型。
- 真值参数两组：
  - Model 1: `β = (2, 0, 0, 4, 0)`
  - Model 2: `β = (2, 9, 0, 4, 8)`
- 候选模型：包含截距的全部 16 个子集模型。
- 噪声方差：`σ^2 = 1` 和 `4`。
- 主要复现先做高斯噪声（与主文一致），t(3) 作为扩展复现实验。

交付物：`config/sim1_config.yml`（记录上述固定参数）。

## 2. 实验骨架实现与校验
- 统一 5-fold 切分函数，固定随机种子，输出 fold 索引。
- 实现四类方法最小闭环：
  - cv
  - cvc（从 `A_cv` 选最简约模型）
  - bic
  - 论文提到的其他基线（fence、F-test）按可实现性分批接入。

验收：先在 n=40、10 次重复下跑通全流程，无报错且输出结构完整。

## 3. CVC 核心模块对齐检查
- 核查点：
  - 使用学生化 max 统计量。
  - 使用 multiplier bootstrap（B=200）。
  - 使用 inequality screening，`α' = 0.005`。
  - `α` 分别取 0.01/0.05/0.1/0.2。
- 对每个候选模型输出：`p` 值、是否进入 `A_cv`、最终是否被选中。

验收：单次运行可导出候选级明细表（便于审计计算正确性）。

## 4. 正式复现实验运行
- 样本量：40/80/160/320/640。
- 真模型：Model 1、Model 2。
- 噪声方差：1、4。
- 每个组合重复 100 次。

输出表：
- 正确选模率（每方法）。
- CVC 置信集平均大小/中位数。
- 运行时间统计（可选）。

## 5. 与论文 Table 1 对照
- 按 `α` 分组生成对照表：
  - 行：Model × `σ^2`
  - 列：n
  - 值：正确选模率
- 增加“偏差列”：复现值 - 论文值。

判定规则：
- 主判定看趋势一致性。
- 若某些格点偏差较大，优先排查：
  1. fold 切分是否一致可复现；
  2. 候选模型枚举是否完整；
  3. CVC 筛选阈值与 p 值方向是否实现正确。

## 6. 风险与缓解
- 风险1：R/包版本差异导致数值偏差。
  - 缓解：锁定 `renv`，记录 sessionInfo。
- 风险2：fence 或 F-test 方法实现细节复杂。
  - 缓解：先完成 cv/cvc/bic 主体复现，再补齐其余基线。
- 风险3：计算时间偏长。
  - 缓解：先小重复数冒烟测试，再扩展到 100 次。

## 7. 建议执行顺序（最稳妥）
1. 先完成 `n=40`、10次重复的端到端冒烟。
2. 扩展到全部 n，但只跑 Model 1 + `σ^2=1`。
3. 全组合 100 次完整跑。
4. 最后接入扩展噪声（t(3)）与额外基线。
